总结

优点 : 进行增加、修改、移动时代价很低; 查询自己、直接下级非常方便
缺点 : 如若需要使用层级结构，例如获取所有子目录，所有下级，代价趋近∞（致命）
适用 : 不涉及“所有子集”，严格按照层级一层层地查询
